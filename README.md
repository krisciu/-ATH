# ~ATH

> A terminal that tells stories.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/krisciu/tildeath/main/install.sh | bash
```

You'll need an Anthropic API key: https://console.anthropic.com/

## Run

```bash
tildeath
```

## Note

Go in blind if you can. The story is better that way.
